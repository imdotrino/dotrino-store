Dotrino — Message Store

This page is the cross-instance storage for Dotrino message threads. It is meant to be embedded as an invisible iframe by the messenger and similar apps.

All data lives in IndexedDB at this origin only (persistent, large quota) — never leaves your browser and never reaches any server. Optional end-to-end encrypted sync to your own Google Drive is available but off by default.

Source: github.com/imdotrino/dotrino-store
